Logical Model: Patienten-Stammdaten

Official URL: https://gabriel0316.github.io/ig-tooling-pages/ig/ig-tooling/StructureDefinition/PatientHI Version: 0.2.0
Draft as of 2024-04-05 Computable Name: PatientHI

Patient/Teilnehmer basierend dem Entwurf der Datenspezifikation des modularen Rahmenkonzepts für Österreich für das Disease-Management bei chronischer Herzinsuffizienz.

Usage:

  • Use this Logical Model Profile: Medizinische Datensätze - Erstdokumentation (im Rahmen der Erstabklärung erhoben/durchgeführt) and Medizinische Datensätze - Folgedokumentation
